# 🛒 Retail Customer Behavior Analytics

## 📝 Overview
This end-to-end data analytics project analyzes 3,900 retail transactions to uncover actionable business insights. The pipeline covers the entire data lifecycle: from data cleaning and feature engineering in Python, to advanced querying in PostgreSQL, and finally, building an interactive dashboard in Power BI to drive business strategy.

## 🛠️ Tools & Technologies Used
* **Python (Pandas):** Data cleaning, missing value imputation, and feature engineering (e.g., segmenting age groups, standardizing column names).
* **PostgreSQL:** Advanced structured data analysis, utilizing subqueries, aggregations, and window functions to answer critical business questions.
* **Power BI:** Created an interactive, client-ready dashboard to visualize KPIs like total revenue, customer segmentation, and sales by category.

## 🔑 Key Business Insights Discovered
* **Demographic Value:** Identified which age groups and genders drive the most revenue.
* **Customer Segmentation:** Categorized customers into New, Returning, and Loyal brackets based on purchase frequency to assess the success of subscription models.
* **Product Performance:** Ranked top products based on average review ratings and analyzed which items rely most heavily on applied discounts to sell.
* **Shipping Optimization:** Compared average purchase amounts between standard and express shipping to inform delivery infrastructure investments.

## 📂 Repository Contents
* `Customer_Shopping_Behavior_Analysis.ipynb`: The Jupyter Notebook containing the initial Python data manipulation and database connection code.
* `customer_behavior_sql_queries.sql`: The full suite of PostgreSQL queries used for in-depth analysis.
* `customer_shopping_behavior.csv`: The raw dataset used for the analysis.
* `customer_behavior_dashboard.pbix`: The interactive Power BI dashboard file.
